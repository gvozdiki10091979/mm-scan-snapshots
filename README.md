# MM Scan Public Snapshots

Public aggregates and signal data from MM Scan trading system.
Updated every 6 hours via cron.

## Files
- `snapshots/latest.md` - Human-readable snapshot (markdown)
- `snapshots/latest.json` - Structured data (JSON)
- `snapshots/history/` - Historical snapshots (6h intervals)

## What's published
- TP rate aggregates (live + full historical)
- Pre-reg verdicts (H016, milestones) - frozen
- Last 50 signals (ticker, entry, finals)
- Listener health status

## What's NOT published
- Detector logic / code
- Pre-reg framework methodology
- Trading bot internals

Generated by snapshot pipeline (Phase 4, 2026-06-12).
